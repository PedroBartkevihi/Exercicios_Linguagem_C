# Exercícios em C

Este repositório reúne exercícios desenvolvidos durante meus estudos de programação em linguagem C.

O objetivo é praticar os principais fundamentos da linguagem, registrar minha evolução e manter um histórico organizado das listas resolvidas.

## Conteúdos estudados

### Lista 1 — Variáveis, entrada e saída, expressões aritméticas e conversão de tipo

Exercícios focados em:

- Declaração de variáveis
- Entrada de dados com `scanf`
- Saída de dados com `printf`
- Operações aritméticas
- Conversão de tipos
- Divisão inteira e resto da divisão
- Uso de constantes simples
- Cálculos matemáticos básicos

### Lista 2 — Expressões relacionais e lógicas, estruturas de seleção e funções matemáticas

Exercícios focados em:

- Operadores relacionais
- Operadores lógicos
- Estruturas `if`, `else if` e `else`
- Estrutura `switch`
- Validação simples de entrada
- Uso da biblioteca `math.h`
- Funções como `sqrt()` e `pow()`
- Classificação por faixas e tabelas de decisão

## Como compilar

Para compilar um exercício em C com `gcc`:

`gcc Lista1/Exercicio01.c -o Exercicio01.exe`

Para exercícios que utilizam funções matemáticas da biblioteca `math.h`, pode ser necessário usar:

`gcc Lista2/Exercicio01.c -o Exercicio01.exe -lm`

## Objetivo

Organizar meus estudos, praticar lógica de programação em C e registrar minha evolução como programador.
